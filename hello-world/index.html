Hello World

June 11, 2024

Sudah beberapa bulan ini saya ngulik-ngulik bearblog, mudah dan nyaman banget nulis disini. Bisa login di hp, menu-menunya simple, nulis pakai markdown dll.

Cukup install note yang support markdown (saya pake obsidian, ada di playstore), nulis, buka bearblog di browser, copy paste deh. Kalau kamu awam tapi pengen nge-blog juga, bisa banget pake bearblog.

Tapi masalahnya, dari semua keunggulan itu saya nggak bisa melakukan yang saya pengen. Yaitu pakai subdomain sendiri, note.galuhmaher.com. Kalo mau harus premium.

Berhubung saya masih males-malesan nulis, rugi kalo keluar duit tiap bulan. Kalo ngikut kurs hari ini, $5x16ribu = 80ribu. Untuk Indonesia diskon 50% jadinya 40ribu per bulan.

Saya kan mendang-mending, jadilah saya mencoba hugo. Kebetulan ada tema bearblog juga disitu.

Sejauh ini sih mulai ngerti dikit-dikit (lebih tepatnya dikit banget) tentang struktur hugo. Yang penting ngerti cara buat postingan lah. Nah, tinggal deploy-nya ini yang nggak ngerti-ngerti, git commit git push apalah itu.

Kalau kamu sekarang sedang membaca tulisan ini, berarti saya berhasil nge-deploy hugo. Dan resmi, saya akan fafifu wasweswos di blog ini.
